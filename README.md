## Описание

В ходе выполнения лабораторной работы я изучил основы языка программирования JavaScript, его синтаксис и основные конструкции. Было проведено сравнение с языком C# для выявления сходств и различий.

### Что было изучено:

1. **Основы синтаксиса JavaScript**
   - Отсутствие строгой типизации
   - Динамическая природа языка
   - Способы запуска кода (браузер, Node.js)

2. **Циклы**
   - `for` — счётный цикл
   - `while` — цикл с предусловием
   - `do...while` — цикл с постусловием
   - `for...of` — перебор итерируемых объектов
   - `for...in` — перебор свойств объектов

3. **Функции**
   - Классическое объявление (`function`)
   - Function Expression (функции как значения)
   - Стрелочные функции (arrow functions)
   - Параметры по умолчанию
   - Замыкания (closures)

4. **Массивы**
   - Создание и доступ к элементам
   - Динамический размер
   - Методы: `push()`, `pop()`, `indexOf()`, `includes()`
   - Хранение разнотипных данных

5. **Объекты**
   - Создание без класса (object literal)
   - Доступ к свойствам (через точку и квадратные скобки)
   - Добавление и удаление свойств
   - Методы объектов
   - Вложенные объекты и массивы


### Отличия JavaScript от C#:

| Характеристика | JavaScript | C# |
|----------------|------------|-----|
| Типизация | Динамическая | Статическая |
| Создание объектов | Без классов (object literal) | Требуется класс |
| Типы данных в массиве | Могут быть разными | Должны быть одинаковыми |
| Добавление свойств | Можно в любой момент | Нельзя без изменения класса |
| Запуск | Браузер / Node.js | Компиляция в IL |
